# zenitha-api

This is the API for Zenitha productivity app.

## Requirements

-   `Node Version` is in `.nvmrc`.

### Run Locally

```bash
    yarn dev
```

### Build

```bash
    yarn build
```

### Testing

```bash
    yarn test
```

### Testing Coverage

```bash
    yarn test:coverage
```

## Build Development Docker Image

```bash
    docker-compose -f docker-compose.yml -f docker-compose.dev.yml up -d
```
